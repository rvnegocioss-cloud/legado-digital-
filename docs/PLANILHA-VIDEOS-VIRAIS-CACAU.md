## 🎯 DICAS DE OURO

### Para viralizar:
1. **Primeiros 3 segundos são TUDO** - hook forte ou perde o viewer
2. **Música é 50% do sucesso** - usar trending sounds
3. **Textos na tela** - muitos watchers assistem sem som
4. **Engaje nos comments** - responde 10+ comments por vídeo
5. **Posta todo dia** - consistência > perfeição
